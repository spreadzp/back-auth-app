 # Back-end with jwt token, Momgoose and Handlebars view templates

## install dependencies
```
npm install -g redis-server
npm i
```

## create .env and fill secret data
```
cp env .env
```

## start redis server
```
redis-server
```
## in new termnal start app
```
npm start
```
